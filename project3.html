WalkCycle GIF

Skills

  • Animation

Programs

  • Photoshop

Description

A class assignment to animate a looping GIF in Photoshop.

This was the final project in Photoshop Raster Graphics class. Since the GIF was require to loop and I needed to strength my animation fundamentals, I chose to animate a walk cycle. The goals of this project was to create a smooth, looping cycle where the character appeared to have some amount of weight and 3-dimensionality. The hood of the character was designed to help with this, as well as add personality and secondary action.
